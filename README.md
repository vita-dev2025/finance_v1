# Finance - Sistema de Controle Financeiro

Sistema pessoal para controle financeiro desenvolvido com HTML, CSS, JavaScript, Node.js e PostgreSQL.

## Funcionalidades

- Dashboard financeiro
- Controle de transações por período
- Categorização de entradas e saídas
- Interface responsiva

## Tecnologias

- Frontend: HTML5, CSS3, JavaScript
- Backend: Node.js, Express
- Banco de Dados: PostgreSQL (Neon.tech)
- Deploy: Render

## Configuração

1. Clone o repositório
2. Instale as dependências: `npm install`
3. Configure as variáveis de ambiente no arquivo `.env`
4. Execute: `npm start`

## Estrutura do Banco

- tbl_periodos: Controle de períodos financeiros
- tbl_transacoes: Registro de transações
- tbl_tp_transacoes: Tipos e categorias de transações
